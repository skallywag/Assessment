Welcome to my Web-site, Trip to Mars. 
It is a website where you can book an affordable trip to the red planet.

Features of the website include: 
A weather app where you can enter a zip code(USA only) and get current weather conditions.
The user can sign up and create an account.
Users can then login to their account and are able to book a trip to Mars.
User can update and confirm trip before booking.
Users can pull current and previous trip history and are able to delete trips.
